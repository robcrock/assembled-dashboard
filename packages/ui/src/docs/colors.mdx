# Colors — the palette

Every color in `packages/ui/src/styles/globals.css`, all in **oklch**, shown live —
the swatches read the actual CSS variables, so this page is always the shipped
truth. This is the reference; the governance lives under rules: how the tiers
relate and why primitives generate no utilities is
[token-tiers](?path=/docs/rules-token-tiers--docs), and where each color may be
*spent* is the [color law](?path=/docs/rules-color-law--docs).

## Primitives

The raw material — shown for reference only: **no Tailwind utilities exist for
these, by design** (the `@theme` exclusion — see
[token-tiers](?path=/docs/rules-token-tiers--docs)). Anchor steps carry the
provenance; everything else is OKLab-generated around them. Primitives are never
themed, so these swatches do not remap with the toolbar toggle — that's the tier
working as designed.
